Sistema de Controle
de Animais Silvestres

Cadastro, listagem, busca e remoção de animais via terminal

🖥️ Menu do Sistema

CONTROLE DE ANIMAIS SILVESTRES
[1] Cadastrar Animal
[2] Listar Animais
[3] Buscar Animal
[4] Remover Animal
[0] Sair

⚙️ Funcionalidades

📝
Cadastrar
Registra um novo animal com ID, nome, espécie, local e status.
📋
Listar
Exibe todos os animais salvos no banco de dados em formato de tabela.
🔍
Buscar
Localiza um animal específico pelo nome de forma rápida.
🗑️
Remover
Exclui um registro pelo ID, atualizando o banco de dados automaticamente.

📁 Estrutura de Arquivos

animais_silvestres/  ← pasta do projeto
 │
 ├── main.c         ← frontend: menu e interação
 ├── animal.h        ← estrutura de dados do animal
 ├── animal.c        ← backend: lógica e funções
 └── dados.txt       ← banco de dados (gerado pelo sistema)

🚀 Compilar e Rodar

$ gcc main.c animal.c -o sistema
// compila todos os arquivos e gera o executável
$ .\sistema.exe
// executa o programa no Windows
$ ./sistema
// executa o programa no Linux / Mac
